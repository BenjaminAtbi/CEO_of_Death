# CEO_of_Death

### Features

#### Disable Lich Skin
Removes the ashen skin effect of the lich path. Requires full game reload to take effect.

If you're a human and still want to look somewhat pallid, you can use [Visual Adjustments 2](https://github.com/BarleyFlour/VisualAdjustments2) to change your racial appearance to Dhampir without changing your actual race.
